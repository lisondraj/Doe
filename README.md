# Doe

Next.js App Router project. The marketing / waitlist experience lives at the site root (`/`).

## Structure

- `app/page.tsx` — home route (client component, Tailwind)
- `app/layout.tsx` — root layout, **Lora** via `next/font/google`
- `app/globals.css` — Tailwind entry

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
