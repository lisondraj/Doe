## Using it in a Next.js app

1. Create or open a Next.js App Router project (with **Tailwind CSS** configured).
2. Copy `app/waitlist/page.tsx` into that project under `app/waitlist/page.tsx`.
3. Ensure **Lora** is available (e.g. `next/font/google` in `app/layout.tsx` or your global CSS).

This folder only contains the duplicated route file; it is not a full runnable Next.js project by itself unless you add `package.json`, `next.config`, `app/layout.tsx`, Tailwind config, etc.
